# Código y Creatividad en el Aula Digital

La mentoría técnica no se trata solo de enseñar sintaxis, sino de cultivar una mentalidad de ingeniería. En mis recientes **Master Classes**, hemos explorado cómo herramientas como **Git** no solo sirven para guardar código, sino para fomentar la colaboración creativa.

## El Caso SEITH

Durante el desarrollo de **SEITH**, un sistema experto basado en Python, mis alumnos se enfrentaron al reto de separar la lógica de negocio de la interfaz de usuario. 

### Lecciones clave:
1. **Arquitectura MVC**: Fundamental para la escalabilidad.
2. **Control de Versiones**: Git como bitácora de decisiones técnicas.
3. **Sistemas de Inferencia**: Cómo traducir el conocimiento humano a reglas de Python.

Este enfoque práctico es lo que busco proyectar en mi camino hacia el **Microsoft MVP**, demostrando que la tecnología es el puente entre la teoría académica y la solución de problemas reales.
